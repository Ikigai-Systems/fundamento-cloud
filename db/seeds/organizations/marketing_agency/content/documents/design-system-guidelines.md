# BrightPath Media - Design System Guidelines

## Overview

This document defines the visual identity standards for BrightPath Media's own brand. All internal materials, social posts, pitch decks, and client-facing documents should follow these guidelines. Consistency builds trust -- with our clients and with prospective clients evaluating our work.

## Logo Usage

- Use the full "BrightPath Media" logo on formal documents and proposals
- The "BP" monogram is acceptable for social media avatars and small spaces
- Minimum clear space: the height of the "B" on all sides
- Never stretch, rotate, recolor, or add effects to the logo
- Logo files are in the shared drive under `/brand-assets/logos/`

## Color Palette

### Primary Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Coral | #FF6B6B | Primary accent, CTAs, highlights |
| Navy | #2C3E50 | Headlines, body text, backgrounds |
| White | #FFFFFF | Backgrounds, text on dark surfaces |

### Secondary Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Light Gray | #F7F9FC | Background surfaces, cards |
| Medium Gray | #95A5A6 | Secondary text, borders |
| Warm Sand | #F5E6D3 | Accent backgrounds, callout boxes |

Use coral sparingly -- it is an accent, not a background color. Navy and white should do the heavy lifting in most layouts.

## Typography

**Headings**: Inter (semi-bold or bold). Use sentence case, not title case, for headings in documents.

**Body text**: System font stack (-apple-system, BlinkMacSystemFont, Segoe UI, sans-serif). 16px base size, 1.5 line height for readability.

**Code and data**: Use a monospace font (JetBrains Mono or system monospace) for any data tables, code snippets, or technical content.

## Social Media Templates

We maintain a set of branded templates for our own social presence:

- **Quote cards**: Navy background, white text, coral accent line
- **Team spotlights**: Photo with coral overlay bar, name and title in Inter
- **Case study highlights**: Split layout with metric on the left, context on the right
- **Blog/article shares**: Image with navy gradient overlay and white headline text

Templates are in Figma. Ask Marcus for access if you do not have it.

## File Naming Conventions

Use lowercase, hyphens for spaces, and include the date where relevant:

```
brightpath-[type]-[description]-[date].[ext]
```

Examples:
- `brightpath-proposal-technova-20260215.pdf`
- `brightpath-social-template-quote-card.fig`
- `brightpath-logo-full-color.svg`

## Slide Deck Standards

- Use the BrightPath slide template (Figma or Google Slides version)
- One idea per slide. Do not overcrowd.
- Maximum 6 bullet points per slide, each under 10 words
- Include the BP monogram in the bottom corner of every slide
- End every deck with a clear next-steps slide and contact information

## When Working on Client Brands

These guidelines apply to BrightPath's own materials only. Each client has their own brand guide -- always reference the client-specific voice and visual guide before creating content for them. When in doubt, check the relevant project space.
