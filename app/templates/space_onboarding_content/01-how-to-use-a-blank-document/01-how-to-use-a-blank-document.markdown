# Creating a document

In order to start writing/typing something, just click orange "Edit" button above

![BlancDoc.jpg](attachment:407)

Newly created Document is in a `DRAFT` mode.

Any edit made to it is being instantly reflected to anyone else visiting that Document.

If you change a text from: "Paragraph" into "Heading1 or 2 or 3" you'll see that Structure of the document is being created (on the top-right)

![Blanc doc.jpg](attachment:408)

This allows rapid collaboration with other people on the initial versions of Documents.

Until you or someone else don't click "Update" button, the document will stay in the Draft mode.

At any time when you acknowledge a document as one that can be shown to others, you can save current version of your document using orange `Update` button

![Document1.jpg](attachment:409)

More about "Draft version" of your document:

Your Team (see "Team" section) will be able to see it, make changes in it, and even if they do not click "Update" button, everyone can see what they have added/changed.

During typing you can also view actual/the latest version of the document using link provided: "View current version", don't worry, changes you have been made so far, are being stored and when you click once more "Edit" button you will see it (dać filmik)

When you click "Update" - from now on, everyone visiting this Document will be directed to the most recent document Version in read-only mode. For further alter/reerange the contents of the Document users need to explicitly enter edit mode again by clicking `Edit` button.

![Document updated.jpg](attachment:410)

From now on everyone also can comment this document not interfering in the main section of the document. Comments will be shown in the separate section of the document (below main section and content of the document)

As you click "Comments" button you'll be able to create a comment.

![Comment1.jpg](attachment:411)

As you write a comment, you have to click "Add" button in order other to see it.

![Comment2.jpg](attachment:412)

As you can see, the comment is being published in the separate section of the document. Everyone can from now on also use an emoji showing their attitude towards the comment.

![Comment3.jpg](attachment:413)

You can create as many document Versions as you wish. To view Historical version of the document use `•••` menu:

With button "+" you can call up different features as: headings, basic blocks, media, emoji and advanced components like charts, formulas, or advanced tables.

**Headings** - use it when you want to quickly create a table of contents, change the document organization and format its design, make it easier for users to browse the document, highlight fragments of the document.

![Headings2.jpg](attachment:419)

`Heading 1`: The largest heading, can be easily added with shortcut `/h1` or CTRL+ALT+1

`Heading 2`: The medium-sized heading, can be easily added with shortcut `/h2` or CTRL+ALT+2

`Heading 3`: The smallest heading, can be easily added with shortcut `/h3` or CTRL+ALT+3

**Basic blocks** - use it when you want to can customize the look of your document by defining your own numbered, bulleted, and check lists.

![Basic Blocks.jpg](attachment:420)

`Numbered List`: Indents your list and automatically generates the next number.

`Bullet List`: Create part of a text as a bullets. Shortcut `*` + `space` (or `-` + `space`, or `+` + `space`)

`Check List`: Checkboxes for tasks and similar usage. Shortcut `[]`

`Paragraph`: It's just your casual plain text.

`Code`: You can add a block of code in any language, complete with syntax coloring.
